
# Welcome to Rezq App

Rezq is a mobile application built with React Native and Expo, designed as a handyman marketplace for the Iraqi market.

The app connects users with skilled workers such as electricians, plumbers, and other service providers in a simple and user-friendly way.

It is fully designed in Arabic with RTL support to match local user needs and provide a smooth and natural experience.

Features

- User authentication using Firebase Authentication  
- Real-time data and storage with Firebase  
- Search and browse service providers  
- Contact forms between users and workers  
- Subscription system using Apple In-App Purchases  
- Push notifications for updates and engagement  
- RTL (Right-to-Left) Arabic support  



# Tech Stack

- React Native (Expo + TypeScript)  
- Firebase (Authentication, Firestore, Cloud Functions, Storage)  
- REST APIs integration  
- Expo Notifications  
- Apple In-App Purchases  



# Project Status

The app is currently in TestFlight and is fully functional.

Due to the current situation in the Middle East, it has not been released yet.  
The next step is to launch the Android version and expand the platform.
